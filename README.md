DEPLOYMENT FOR iPAD:
--------------------

METHOD 1: Direct Safari Access (Easiest)
1. Create a folder on your computer called "Max-Boss-Battle"
2. Put index.html in this folder
3. Create an "assets" subfolder inside
4. Put boss.png, minion1.png, and minion2.png in the assets folder
5. Upload the entire folder to iCloud Drive, Dropbox, or Google Drive
6. On your iPad, open the cloud service app
7. Navigate to the folder and tap index.html
8. It will open in Safari - you can now play!
9. Add to Home Screen: Tap the Share button ‚Üí "Add to Home Screen" for easy access


METHOD 2: Using the Files App
1. Connect your iPad to your computer
2. Open Finder (Mac) or iTunes (Windows)
3. Select your iPad
4. Go to the Files tab
5. Drag the "Max-Boss-Battle" folder to "On My iPad"
6. On iPad, open the Files app
7. Navigate to "On My iPad" ‚Üí "Max-Boss-Battle"
8. Tap index.html to play


METHOD 3: GitHub Pages (For Online Access)
1. Create a GitHub account at github.com
2. Upload all files to a repository
3. Go to Settings ‚Üí Pages
4. Enable GitHub Pages
5. Your game will be available at: https://yourusername.github.io/repositoryname
6. Access this URL from any device


METHOD 4: Using Documents by Readdle App (Recommended)
1. Download "Documents by Readdle" from App Store (free)
2. Transfer your Max-Boss-Battle folder to the iPad via:
   - iCloud Drive
   - iTunes File Sharing
   - WiFi Transfer in Documents app
3. In Documents app, navigate to the folder
4. Tap index.html
5. The game opens in the built-in browser
6. Works offline!

TROUBLESHOOTING:
----------------
- Images not showing? Make sure the assets folder is in the same location as index.html
- Game won't start? Make sure you're using a modern browser (Safari, Chrome, Firefox)
- Touch not working? Try tapping the input field first
- Want to reset progress? Clear browser data or tap "Play Again"


TIPS FOR TEACHERS/PARENTS:
---------------------------
- Game saves progress automatically
- Students advance through difficulty levels as they improve
- Incorrect answers reduce HP but don't end the game immediately
- Streak counter encourages consecutive correct answers
- Boss battles every 10 questions add excitement
- All content is age-appropriate for grades 2-5
